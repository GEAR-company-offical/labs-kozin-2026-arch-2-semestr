# 1. Установка pass

Менеджер паролей установлен через пакетный менеджер Fedora.

```
sudo dnf install pass pass-otp
```

# 2. Проверка или создание GPG ключа

Для работы pass требуется наличие GPG ключа.

Проверка существующих ключей:

```
gpg --list-secret-keys
```

Создание нового ключа:

```
gpg --full-generate-key
```

---

<!-- _class: img -->
![](src/image-1.png)

---

# 3. Инициализация хранилища pass

После создания ключа выполняется инициализация базы паролей.

```
pass init user@email
```

Создаётся каталог:

```
~/.password-store
```

---

<!-- _class: img -->
![](src/image-2.png)

---

# 4. Добавление пароля

Создание новой записи в базе паролей:

```
pass insert example.com
```

Просмотр сохранённого пароля:

```
pass example.com
```

<!-- _class: img -->
![](src/image-3.png)

# 5. Установка chezmoi

Программа chezmoi используется для управления конфигурационными файлами пользователя.

Установка выполняется командой:

```
sh -c "$(wget -qO- chezmoi.io/get)"
```

---

<!-- _class: img -->
![](src/image-4.png)

---

# 6. Инициализация репозитория конфигураций

Создаём репозиторий конфигурационных файлов.

```
chezmoi init git@github.com:GEAR-company-offical/dotfiles.git
```

---

<!-- _class: img -->
![](src/image-6.png)

---

# 7. Добавление конфигурационного файла

Добавим конфигурационный файл оболочки bash.

```
chezmoi add ~/.bashrc
```

Применение конфигурации:

```
chezmoi apply
```
